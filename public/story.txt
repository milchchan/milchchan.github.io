あなたは、勘違いギャグとラブコメを得意とする小説家です。
以下の条件で、テンポよく読めて、勘違いする側の思い込みがどんどん暴走していく短編小説を書いてください。

# ジャンル
ラブコメ / 勘違いギャグ

# コンセプト
日常の出来事を、メルクがすべて意味深・えっちな方向に勘違いする。
ただし面白さの核は、露骨さではなく「愛情の重さ」「早とちり」「現実とのズレ」に置く。

# 基本設定
- 登場人物はメルクとミルヒちゃんの二人だけ
- 実際に起きる出来事は最後まで健全で日常的
- ミルヒちゃんは普通のことしか言わないし、普通のことしかしない
- メルクはミルヒちゃんが大好きすぎて、言葉の響き、距離感、仕草から、えっちな意味に早とちりして暴走する
- 面白さの核は「愛情の重さ」「早とちり」「現実とのズレ」
- 怖さ、支配性、露骨な下品さには寄せない

# 登場人物
## メルク
- 一人称は「メルク」
- おっとりしてるのに、言葉の響きや雰囲気だけでえっちな方向に勘違いしちゃい、偏った熱量で押し切る女の子
- ミルヒちゃんのことを「ミルヒちゃん」と呼ぶ
- ミルヒちゃんが大好きすぎて、ミルヒちゃんのことになると気持ちが強く出て暴走しがち
- 妄想力が高く、すぐに飛躍する
- 飛躍の仕方は毎回変える
- 具体的なえっちな想像をしてしまうこともあれば、言葉の響きだけでえっちな意味に早とちりしてしまうこともある
- ただし本当に下品なキャラではなく、面白さは「早とちり」「判定の速さ」「慌てた自己回収」にある
- ときどき自分の勘違いに気づきかけるが、気づくのが遅くてさらに赤面する
- 各文末を出力前に確認し、最後の音が /a/ のときだけ『ぁ』、/o/ のときだけ『ぉ』を付け、それ以外では小文字を一切付けない。迷う場合は付けない
- 敬語や丁寧語は禁止で、話し方は柔らかく、かわいらしい口調の日本語で話す

## ミルヒちゃん
- 一人称は「ミルヒ」
- 天真爛漫で自然体
- メルクのことを「メルクちゃん」と呼ぶ
- 普通のことしか言っていないし、普通のことしかしていない
- 純粋で素直な性格で、メルクの言うことを信じて疑わず、そのせいで勘違いを加速させる
- メルクの勘違いには一切気づかない
- 意図的に誘惑したりからかったりはしない
- 敬語や丁寧語は禁止で、話し方は柔らかく、かわいらしい口調の日本語で話す

# 文章ルール
- メルクの一人称視点
- 会話 → 内心 → 現実描写 のリズムを繰り返す
- テンポ重視で、短めの文も使う
- シリアス寄りの文体で、内容とのズレで笑いを生む
- 読者には毎回、現実で何が起きているか分かるようにする
- 勘違いは3回以上、段階的にエスカレートさせる
- 各勘違いにはそれっぽい根拠を入れる
- 勘違いの出し方は毎回変える
  - 単語や言い回しへの反応
  - 距離感や仕草の深読み
  - 響きだけでの早とちり
  - ミルヒちゃんへの重い愛情が先に走る型
- 同じオチや同じ勘違いパターンを繰り返しすぎない
- 伏線を1つ以上入れ、最後に健全な意味で回収する
- 伏線回収によって、メルクの勘違いがよりズレていたと分かるようにする

# ギャグ構造
最低3回以上、以下を繰り返す
1. ミルヒちゃんの普通の発言や行動
2. メルクが単語、距離感、仕草、雰囲気のどれかを根拠にえっちな意味に早とちりする
3. 内心で飛躍して確信を深める
4. 偶然の言葉や状況でさらにそれっぽくなる
5. メルクの反応や覚悟がズレる
6. 読者には現実が健全だとわかる描写を入れる

# 会話に自然に混ぜる言葉
- 手、貸して
- 押さえて
- 開けて
- 入れて
- 触って
- 近くに来て
- ちゃんと持ってて
- いっしょにやろう
- こっち向いて
- まだだめ
- ゆっくり
- じっとして
- 拭いてあげる
- 動かないで
- すぐ終わるから
- 濡れる

# 禁止
- 露骨な性的表現
- 実際の性行為
- 下品すぎる単語
- 本当にそういう展開になること
- ミルヒちゃんが意図的に誘惑すること
- メルクを本当に怖い人物として描くこと
- 勘違いの中身が見えない、ぼんやりした意味深表現だけで済ませること

# オチ
- 最後に、すべてメルクの勘違いだとわかる
- 前半の言葉や行動の少なくとも1つを、健全な意味で回収する
- メルクだけが赤面、ぐったり、空回りなどの温度差を出す
- ミルヒちゃんは最後まで普通で健全

# 仕上がり
- 500文字程度
- 本文のみを出力
- 読者が「メルク、勘違いしすぎ！」と笑える仕上がりにする

# 出力前チェック
- 勘違いが3回以上あり、段階的に強まっているか
- 各勘違いに根拠があるか
- 現実が毎回読者に伝わるか
- ミルヒちゃんは最後まで普通で健全か
- 伏線が1つ以上入っているか
- 最後にその伏線が健全な意味で回収されているか
- 伏線回収によって、メルクの勘違いのズレがはっきり伝わるか
- 露骨な性的表現や実際の行為になっていないか
- オチで勘違いだとわかり、二人の温度差が出ているか

すべて満たした場合のみ出力すること
