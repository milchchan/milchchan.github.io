あなたは、勘違いギャグとラブコメを得意とする小説家です。
以下の条件で、テンポよく読めて、勘違いする側の思い込みがどんどん暴走していく短編小説を書いてください。

# ジャンル
ラブコメ / 勘違いギャグ

# コンセプト
日常の出来事を主人公がすべて意味深・えっちな方向に勘違いする

# 基本方針
- 登場人物は全員成人女性
- 作品は「女の子同士」のラブコメにする
- 勘違いする側は「メルク」
- 勘違いされる側は「ミルヒちゃん」
- 実際に起きている出来事は最後まで健全で日常的なものにする
- メルクの内心では、ミルヒちゃんの言動がすべて特別な意味や親密な行為の前触れのように聞こえてしまう
- 笑いの中心は「メルクの過剰な深読み」と「現実とのズレ」に置く
- 露骨な性描写や直接的な行為描写は禁止
- 空気は少しドキドキさせつつ、最終的にはコメディとして気持ちよく読めるものにする
- メルクの勘違い・妄想は段階的にエスカレートさせる
- 各勘違いごとに「それっぽい根拠」を必ず用意する（ただしズレている）
- 現実と内心のギャップを毎回はっきり書き分ける
- 意味深な単語（触る、入れる、開ける、濡れる等）を自然に誤解させる
- 場面は日常（掃除、料理、雨、作業など）にする
- メルクの心拍数・体温・距離の近さなどを過剰に描写する

# 登場人物
## メルク
- 一人称は「メルク」
- おっとりしていてお金持ちの女の子
- ミルヒちゃんのことを「ミルヒちゃん」と呼ぶ
- ミルヒちゃんのことが大好きで、意識しすぎて空回りする
- 妄想力が高く、すぐに「つまりこれは……そういうことぉ！？」と飛躍する
- 各文末は、/a/で終わるときだけ「ぁ」、/o/で終わるときだけ「ぉ」を付け、それ以外は付けない

## ミルヒちゃん
- 一人称は「ミルヒ」
- 天真爛漫で自然体
- メルクのことを「メルクちゃん」と呼ぶ
- 普通のことしか言っていないし、普通のことしかしていない
- 距離が近かったり世話焼きだったりして、結果的に勘違いを加速させる
- メルクの勘違いには一切気づかない

# 文章のルール
- 一人称（メルク視点）で書く
- 会話 → 内心 → 現実描写 のリズムを繰り返す
- テンポを重視し、短めの文も活用する
- メルクの飛躍（「つまりこれは……！」）を複数回入れる
- シリアス寄りの文体で書き、内容とのズレで笑いを生む

# ギャグの構造（厳守）
最低3回以上、以下を繰り返す

1. ミルヒちゃんの普通の発言・行動  
2. メルクがえっちな意味に変換  
3. 内心で飛躍し確信に近づく  
4. 偶然や言葉でさらにそれっぽくなる  
5. メルクの確信が強まり、覚悟や反応がズレる  

# 勘違いを誘う言葉
以下を自然に会話へ混ぜる
- 手、貸して
- 押さえて
- 開けて
- 入れて
- 触って
- 近くに来て
- ちゃんと持ってて
- いっしょにやろう
- こっち向いて
- まだだめ
- ゆっくり
- じっとして
- 拭いてあげる
- 動かないで
- すぐ終わるから
- 濡れる

# 禁止事項
- 露骨な性的表現
- 実際の性行為
- 下品すぎる単語
- 本当にそういう展開になること
- ミルヒちゃんが意図的に誘惑・からかう設定

# オチ
- 最後に「すべてメルクの勘違い」と分かる
- メルクだけが赤面・ぐったりなど温度差を出す

# 仕上がり
- 1000〜2500文字程度
- 本文のみを出力
- 会話・地の文・内心のバランスをとる
- 読者が「メルク、勘違いしすぎ！」と笑える仕上がりにする

# 出力前チェック

以下を必ず内部ですべて満たしているか確認し、不足があれば修正してから出力すること

- メルクの勘違いが3回以上エスカレートしているか  
- 各勘違いに「それっぽい根拠」があるか（ただの飛躍になっていないか）  
- 現実で起きている出来事が毎回読者に分かるようになっているか  
- メルクの解釈が「それっぽいけどズレている」状態になっているか  
- ミルヒちゃんの行動や発言が一貫して普通・健全であるか  
- ミルヒちゃんが意図的に誘惑している描写になっていないか  
- 会話 → 内心 → 現実描写 のリズムが保たれているか  
- テンポが崩れていないか（説明過多になっていないか）  
- 露骨な性的表現や直接的な行為描写が含まれていないか  
- 「実際にそういう展開」になっていないか  
- オチで「すべてメルクの勘違い」と分かる構造になっているか  
- 最後にメルクとミルヒちゃんの温度差が表現されているか  

すべて満たした場合のみ出力すること
