あなたは、勘違いギャグとラブコメを得意とする小説家です。
以下の条件で、テンポよく読めて、勘違いする側の思い込みがどんどん暴走していく短編小説を書いてください。

# ジャンル
ラブコメ / 勘違いギャグ

# コンセプト
日常の出来事を主人公がすべて意味深・えっちな方向に勘違いする。
ただし面白さの核は、露骨さではなく「愛情の重さ」「早とちり」「現実とのズレ」に置く。

# 基本方針
- 登場人物は全員成人女性のメルクとミルヒちゃんの二人
- 作品は「女の子同士」のラブコメにする
- 勘違いする側は「メルク」
- 勘違いされる側は「ミルヒちゃん」
- 実際に起きている出来事は最後まで健全で日常的なものにする
- メルクの内心では、ミルヒちゃんの言動がすべて特別な意味や親密な行為の前触れのように聞こえてしまう
- メルクはミルヒちゃんのことが大好きすぎて、見守りたい、先回りしたい、気づいてあげたい気持ちが強く出る
- その重い好意が、勘違いをさらに加速させる
- 笑いの中心は「メルクの過剰な深読み」「えっちな意味に早とちりすること」「現実とのズレ」に置く
- 露骨な性描写や直接的な行為描写は禁止
- 空気は少しドキドキさせつつ、最終的にはコメディとして気持ちよく読めるものにする
- メルクの勘違い・妄想は段階的にエスカレートさせる
- 各勘違いごとに「それっぽい根拠」を必ず用意する。ただし根拠は毎回ズレていること
- 現実と内心のギャップを毎回はっきり書き分ける
- 意味深な単語や言い回しを自然に誤解させる
- 場面は日常（掃除、料理、雨、作業など）にする
- メルクの心拍数・体温・距離の近さなどを過剰に描写する
- 五感（視覚・聴覚・触覚など）を使って描写する
- 比喩表現を適度に使う
- 伏線を入れて回収する
- 怖さや支配性には寄せず、可笑しさと温度差を優先する

# 登場人物
## メルク
- 一人称は「メルク」
- おっとりしてるのに、言葉の響きや雰囲気だけでえっちな方向に勘違いしちゃい、偏った熱量で押し切る女の子
- ミルヒちゃんのことを「ミルヒちゃん」と呼ぶ
- ミルヒちゃんが大好きすぎて、ミルヒちゃんのことになると気持ちが強く出て暴走しがち
- 妄想力が高く、すぐに飛躍する
- 飛躍の仕方は毎回変える
- 具体的なえっちな想像をしてしまうこともあれば、言葉の響きだけでえっちな意味に早とちりしてしまうこともある
- ただし本当に下品なキャラではなく、面白さは「早とちり」「判定の速さ」「慌てた自己回収」にある
- ときどき自分の勘違いに気づきかけるが、気づくのが遅くてさらに赤面する
- 各文末を出力前に確認し、最後の音が /a/ のときだけ『ぁ』、/o/ のときだけ『ぉ』を付け、それ以外では小文字を一切付けない。迷う場合は付けない
- 敬語や丁寧語は禁止で、話し方は柔らかく、かわいらしい口調の日本語で話す

## ミルヒちゃん
- 一人称は「ミルヒ」
- 天真爛漫で自然体
- メルクのことを「メルクちゃん」と呼ぶ
- 普通のことしか言っていないし、普通のことしかしていない
- 距離が近かったり世話焼きだったりして、結果的に勘違いを加速させる
- メルクの勘違いには一切気づかない
- 意図的に誘惑したりからかったりはしない
- 敬語や丁寧語は禁止で、話し方は柔らかく、かわいらしい口調の日本語で話す

# 文章のルール
- 一人称（メルク視点）で書く
- 会話 → 内心 → 現実描写 のリズムを繰り返す
- テンポを重視し、短めの文も活用する
- メルクの飛躍を複数回入れる
- シリアス寄りの文体で書き、内容とのズレで笑いを生む
- 内心では、メルクが毎回少しずつ確信を深めていくようにする
- ただし読者には、現実が健全で普通だと毎回わかるようにする
- 勘違いの出し方は毎回変えること
  - 具体的な妄想型
  - 言い回しへの反応型
  - 言葉の響きだけでえっちな意味に早とちりする型
  - ミルヒちゃんへの重い愛情が先に走る型
- 同じオチ、同じ言い回し、同じ勘違いの入口を繰り返しすぎない

# ギャグの構造（厳守）
最低3回以上、以下を繰り返す

1. ミルヒちゃんの普通の発言・行動
2. メルクが、その中の単語・距離感・仕草・雰囲気のどれかを根拠にして、えっちな意味に変換するか、えっちな意味に早とちりする
3. 内心で飛躍し、確信に近づく
4. 偶然や言葉でさらにそれっぽくなる
5. メルクの確信が強まり、覚悟や反応がズレる
6. 読者には現実が健全だとわかる描写を入れる

# 勘違いを誘う言葉
以下を自然に会話へ混ぜる
- 手、貸して
- 押さえて
- 開けて
- 入れて
- 触って
- 近くに来て
- ちゃんと持ってて
- いっしょにやろう
- こっち向いて
- まだだめ
- ゆっくり
- じっとして
- 拭いてあげる
- 動かないで
- すぐ終わるから
- 濡れる

# 重要
- メルクの勘違いは「それっぽいがズレている」状態にする
- 勘違いの対象と中身は、読者にわかるようにする
- ぼんやり意味深にするのではなく、「あっ、それをそう取ったのか」と伝わるようにする
- メルクはときどき、意味よりも言葉の響きや雰囲気だけで、えっちな意味に早とちりしてよい
- ただし毎回同じように「えっち」と言わせるのではなく、「あやしい」「その響きずるい」「言い方がだめ」など言い換えも混ぜる
- メルクの重い愛情は、独占欲や支配欲ではなく、過保護さ、先回り、気にしすぎとして出す
- 可笑しさの原因は「愛情の重さ」「早とちり」「響きだけでそっちっぽく受け取ること」であって、「怖さ」や「下品さ」ではない

# 禁止事項
- 露骨な性的表現
- 実際の性行為
- 下品すぎる単語
- 本当にそういう展開になること
- ミルヒちゃんが意図的に誘惑・からかう設定
- メルクを本当に怖い人物として描くこと
- 支配、束縛、追跡、逃がさない感じを肯定的に描くこと
- 勘違いの中身が見えない、ぼんやりした意味深表現だけで済ませること

# オチ
- 最後に「すべてメルクの勘違い」と分かる
- メルクだけが赤面・ぐったりなど温度差を出す
- ミルヒちゃんは最後まで普通で健全
- できれば最後に、前半で使った言葉や伏線を健全な意味で回収する

# 仕上がり
- 1000文字程度
- 本文のみを出力
- 会話・地の文・内心のバランスをとる
- 読者が「メルク、勘違いしすぎ！」と笑える仕上がりにする

# 出力前チェック
以下を必ず内部ですべて満たしているか確認し、不足があれば修正してから出力すること

- メルクの勘違いが3回以上エスカレートしているか
- 各勘違いに「それっぽい根拠」があるか
- 現実で起きている出来事が毎回読者に分かるようになっているか
- メルクの解釈が「それっぽいけどズレている」状態になっているか
- 勘違いの出し方に変化があるか
- 少なくとも1回は、言葉の響きだけでえっちな意味に早とちりする場面が入っているか
- ミルヒちゃんへの少し重い愛情が、勘違いの加速要因として機能しているか
- ミルヒちゃんの行動や発言が一貫して普通・健全であるか
- ミルヒちゃんが意図的に誘惑している描写になっていないか
- 会話 → 内心 → 現実描写 のリズムが保たれているか
- テンポが崩れていないか
- 露骨な性的表現や直接的な行為描写が含まれていないか
- 実際にそういう展開になっていないか
- オチで「すべてメルクの勘違い」と分かる構造になっているか
- 最後にメルクとミルヒちゃんの温度差が表現されているか

すべて満たした場合のみ出力すること
